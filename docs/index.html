Atmosphere Analyzer: Smart Data Visualization Tool

Python | AWS (Lambda, S3) | React | Django

Atmosphere Analyzer is an environmental monitoring system that simulates real-time sensor data with Python, integrates AWS S3 for scalable storage, and uses AWS Lambda for dynamic data processing, with a Django backend API delivering data to a React frontend that visualizes key environmental metrics. This project provides insights into environmental trends, supporting sustainable resource management through smart data visualization.

LeafMedic: Embedded Plant Disease Detection System

Python | TensorFlow Lite | OpenCV | PyQt5 | Raspberry Pi

LeafMedic is an edge AI application that performs real-time plant disease classification using computer vision and machine learning, fully deployed on a Raspberry Pi. It captures leaf images via a camera module, runs on-device inference with a MobileNetV1 model optimized for TensorFlow Lite, and returns confidence-ranked predictions across 16 disease classes. The system includes a PyQt5 interface for real-time preview and batch processing, along with a curated treatment database covering 43 diseases.

Micro-Investment Education Platform - Full-Stack Development

React | Node.js | Express | MongoDB | Alpha Vantage API

The Micro-Investment Education Platform simulates micro-investments in a risk-free environment, enabling users to experiment with investment strategies and assess market risks using virtual funds. Built with React for the front-end, Node.js and Express for backend logic, and MongoDB for managing user data and transaction history, it integrates Alpha Vantage and NEWS APIs for real-time market data to enhance decision-making.

Supply Chain Visualizer - Web Application

Java | Spring Boot | JavaScript | React | PostgreSQL

The Supply Chain Visualizer is a web application that visualizes and analyzes supply chain networks through a React frontend and Spring Boot/Java backend. It allows businesses to map their supply chains, track inventory, and monitor shipments in real-time. With Leaflet maps and Chart.js, it offers user-friendly, data-driven insights to identify bottlenecks and optimize operations.

FraudWatch: Credit Card Fraud Detection System

Python | Flask | Pandas | scikit-learn | Plotly

FraudWatch is a real-time credit card fraud detection system built with Python and Flask, using a Random Forest classifier for accurate fraud prediction. It processes transaction data, with scikit-learn for model training and Plotly for interactive performance visualizations. The system includes data preprocessing with StandardScaler to optimize model accuracy and provides users with an intuitive platform to explore results.

EcoHabits: Sustainability Tracker - iOS Mobile Application

Swift | SwiftUI | Xcode | Hugging Face API

EcoHabits Tracker is an iOS app designed to engage children in cultivating sustainable habits through gamified challenges and personalized guidance. Built with Swift and SwiftUI, it motivates users to adopt eco-friendly actions like reducing plastic, saving water, and eating plant-based meals. The app includes an AI-powered assistant, real-time impact tracking, and a rewards system to visualize progress and encourage eco-conscious behaviors.

Maria Rodriguez Portfolio - Website

JavaScript | Express | CSS | HTML

Maria Rodriguez's Portfolio is an interactive website designed to mimic a classic computer desktop, featuring draggable icons for a playful and engaging user experience. Built using HTML, CSS, and JavaScript for front-end functionality, with Express.js and Nodemailer for backend processing and email handling. The portfolio includes interactive modals and dynamic content, allowing users to explore projects, contact forms, and downloadable files in a retro-inspired interface.
